# sokoban 推箱子
这是一个主要使用threejs框架写的推箱子游戏

使用的主要框架有：
threejs 3D图形的渲染
OrbitControls 3D视角的控制
promise.min.js （仅IE）es6支持，主要用于弹窗美化处理的支持
stats.min.js FPS监控组件
STLLloader 3D模型加载组件
sweetalert 弹窗美化
Buttons 开源按钮样式库
font-awesome 字体图标
